Рациональные выражения

• Рациональная дробь
Рациональная дробь – это дробь вида P(x)/Q(x), где P(x) и Q(x) – многочлены, а Q(x) ≠ 0.
Пример: (x^2 + 3x)/(x - 5).

• Основное свойство рациональной дроби. Сокращение
Если числитель и знаменатель имеют общий множитель, его можно сократить (при условии, что он не равен 0).
Основное свойство: P(x)/Q(x) = (P(x)*k)/(Q(x)*k), где k ≠ 0.
Пример: (6x)/(9x) = 6/9 = 2/3, при x ≠ 0.

• Сложение и вычитание рациональных дробей
Чтобы сложить или вычесть дроби, нужно привести их к общему знаменателю.
Пример: (2/x) + (3/x^2) = (2x/x^2) + (3/x^2) = (2x+3)/x^2, при x ≠ 0.

• Умножение и деление рациональных дробей
Умножение: (a/b) * (c/d) = (ac)/(bd).
Деление: (a/b) : (c/d) = (a/b) * (d/c).
Пример: (2x/3) * (9/4x) = (18x)/(12x) = 3/2, при x ≠ 0.

• Преобразования рациональных выражений
Основные виды преобразований: сокращение, приведение к общему знаменателю, разложение на множители, упрощение сложных дробей.
Пример: (x^2 - 9)/(x^2 - 3x) = ((x-3)(x+3))/(x(x-3)) = (x+3)/x, при x ≠ 0 и x ≠ 3.

Видеоурок

• Пример 1. Сокращение рациональной дроби
Дано: (8x^2)/(12x)
Решение:
 1. Разложим числитель и знаменатель на множители: 8x^2 = 8xx, 12x = 12*x
 2. Сократим общий множитель x и 4: (8xx)/(12x) = (2x)/3
Ответ: (2x)/3, при x ≠ 0

• Пример 2. Сложение рациональных дробей
Дано: (3/x) + (2/x^2)
Решение:
 1. Приведём дроби к общему знаменателю x^2: (3/x) = (3x/x^2)
 2. Складываем числители: (3x + 2)/x^2
Ответ: (3x + 2)/x^2, при x ≠ 0

• Пример 3. Вычитание рациональных дробей
Дано: (5/(x+1)) - (2/(x+1))
Решение:
 1. Дроби уже имеют общий знаменатель: x + 1
 2. Вычтем числители: (5 - 2)/(x+1) = 3/(x+1)
Ответ: 3/(x+1), при x ≠ -1

• Пример 4. Умножение рациональных дробей
Дано: (2x/5) * (15/4x)
Решение:
 1. Умножаем числители и знаменатели: (2x15)/(54x) = 30x/20x
 2. Сократим общий множитель x и 10: 30x/20x = 3/2
Ответ: 3/2, при x ≠ 0

• Пример 5. Деление рациональных дробей
Дано: (3x/4) : (9/8x)
Решение:
 1. Переворачиваем вторую дробь и умножаем: (3x/4) * (8x/9) = (24x^2)/(36)
 2. Сократим общий множитель 12: (2x^2)/3
Ответ: (2x^2)/3, при x ≠ 0

• Пример 6. Преобразование сложного рационального выражения
Дано: (x^2 - 16)/(x^2 - 4x)
Решение:
 1. Разложим на множители: (x-4)(x+4)/[x(x-4)]
 2. Сократим общий множитель (x-4): (x+4)/x
Ответ: (x+4)/x, при x ≠ 0 и x ≠ 4
