Функция числового аргумента. Область определения, множество значений. Способы задания функции

Функция числового аргумента — это соответствие, при котором каждому числу x из некоторого множества X (область определения функции) ставится в соответствие единственное число y из множества Y (множество значений функции).
 • Обозначение: y = f(x), где x ∈ X
 • Область определения функции (D(f)) — множество всех допустимых значений x, для которых функция определена.
 • Множество значений функции (E(f)) — множество всех возможных значений y = f(x).

Способы задания функции:
 1. Аналитический способ — формула, например:
y = 2x + 3, y = x², y = sin(x)
 2. Табличный способ — функция задается таблицей значений:

x -2 -1 0 1 2
y 4 1 0 1 4

 3. Графический способ — функция представляется графиком на координатной плоскости.

⸻

 Свойства функции

Основные свойства функций:
 1. Монотонность:
 • Функция возрастает на промежутке, если для любых x₁ < x₂ выполняется f(x₁) < f(x₂)
 • Функция убывает на промежутке, если для любых x₁ < x₂ выполняется f(x₁) > f(x₂)
 2. Ограниченность:
 • Функция ограничена сверху, если существует число M такое, что f(x) ≤ M для всех x ∈ D(f)
 • Функция ограничена снизу, если существует число m такое, что f(x) ≥ m для всех x ∈ D(f)
 3. Периодичность:
 • Функция периодична, если существует T ≠ 0, такое что f(x + T) = f(x) для всех x ∈ D(f)
 4. Непрерывность и разрывность (тема изучается более подробно в старших классах)

⸻

 Четные и нечетные функции
 1. Четная функция:
 • Функция f(x) называется четной, если f(-x) = f(x) для всех x ∈ D(f)
 • График четной функции симметричен относительно оси Oy
 • Пример: y = x², y = cos(x)
 2. Нечетная функция:
 • Функция f(x) называется нечетной, если f(-x) = -f(x) для всех x ∈ D(f)
 • График нечетной функции симметричен относительно начала координат
 • Пример: y = x³, y = sin(x)

⸻

 Построение графиков функций y = f(x) ≤ b, y = f(x + a)
 1. Сдвиги графика функции:
 • По оси Ox: y = f(x + a) → график сдвигается влево на a, если a > 0, и вправо на |a|, если a < 0
 • По оси Oy: y = f(x) + b → график сдвигается вверх на b, если b > 0, и вниз на |b|, если b < 0
 2. Неравенства с функцией:
 • y = f(x) ≤ b → нужно найти все x, для которых значение функции не превышает b
 • y = f(x) ≥ a → нужно найти все x, для которых значение функции не меньше a
 3. Построение графика по шагам:
 1. Определить область определения функции D(f)
 2. Найти ключевые точки: пересечение с осями, максимумы, минимумы, точки разрыва (если есть)
 3. Использовать сдвиги и отражения, если функция преобразована: y = f(x + a) или y = f(x) + b
 4. Соединить точки плавной линией или кусочно-линейно, если функция дискретная

Примеры с графиками:
 • y = x² + 2 → график параболы y = x² сдвинут вверх на 2
 • y = (x - 3)² → график параболы y = x² сдвинут вправо на 3

Пример 1. Область определения и множество значений функции

Дано: y = √(x - 1)

Найти: область определения D(f) и множество значений E(f)

Решение:
 1. Подкоренное выражение должно быть ≥ 0:
x - 1 ≥ 0 → x ≥ 1
→ D(f) = [1, +∞)
 2. Так как √(x - 1) ≥ 0 для всех x ∈ D(f), множество значений:
E(f) = [0, +∞)

Ответ: D(f) = [1, +∞), E(f) = [0, +∞)

⸻

Пример 2. Четная и нечетная функция

Дано: f(x) = x² - 4

Найти: четность функции

Решение:
f(-x) = (-x)² - 4 = x² - 4 = f(x) → функция четная

Ответ: f(x) — четная функция

⸻

Дано: g(x) = x³ - x

Решение:
g(-x) = (-x)³ - (-x) = -x³ + x = -(x³ - x) = -g(x) → функция нечетная

Ответ: g(x) — нечетная функция

⸻

Пример 3. Построение графика сдвига

Дано: y = (x - 2)² + 3

Решение:
 1. Основная функция: y = x² (парабола с вершиной в (0,0))
 2. Сдвиг по оси Ox на 2 → вершина переместилась вправо: (0,0) → (2,0)
 3. Сдвиг по оси Oy на 3 → вершина переместилась вверх: (2,0) → (2,3)

График: парабола с вершиной в точке (2,3)

⸻

Пример 4. Решение неравенства с функцией

Дано: y = x² - 4 ≤ 0

Найти: x, при которых выполняется неравенство

Решение:
 1. x² - 4 ≤ 0 → x² ≤ 4
 2. -2 ≤ x ≤ 2

Ответ: x ∈ [-2, 2]

⸻

Пример 5. Монотонность функции

Дано: y = -2x + 5

Найти: на каких промежутках функция возрастает или убывает

Решение:
 1. Коэффициент перед x отрицательный: -2 < 0
 2. Значит, функция убывает на всей области определения (-∞, +∞)

Ответ: функция убывает на (-∞, +∞), возрастает нет
