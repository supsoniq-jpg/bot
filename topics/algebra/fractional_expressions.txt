Дробно-рациональные уравнения и неравенства

Дробно-рациональные уравнения — это уравнения, где переменная находится в знаменателе дроби.
Примеры:
1/(x - 2) = 3, (x + 1)/(x² - 4) = 0

Особенность: знаменатель не может быть равен нулю, поэтому сначала находят область допустимых значений (ОДЗ).

Метод решения:
 1. Найти ОДЗ.
 2. Умножить обе части на общий знаменатель.
 3. Решить полученное уравнение.
 4. Проверить, чтобы корни принадлежали ОДЗ.

Пример:
(2x - 3)/(x - 1) = 1
ОДЗ: x ≠ 1
Умножаем обе части на (x - 1): 2x - 3 = x - 1 → x = 2
Ответ: x = 2

⸻

Системы нелинейных уравнений — это системы, в которые входит хотя бы одно нелинейное уравнение (с квадратами, корнями или дробями).

Пример:
{
x² + y² = 25
y = x + 1
}

Методы решения:
 • подстановки (выражаем одну переменную и подставляем),
 • сравнения (выражаем одну переменную из двух уравнений и приравниваем),
 • графический метод (ищем точки пересечения графиков).

⸻

Формула длины отрезка и уравнение окружности.
Если даны точки A(x₁, y₁) и B(x₂, y₂), то
AB = √((x₂ - x₁)² + (y₂ - y₁)²).

Уравнение окружности с центром O(a, b) и радиусом R:
(x - a)² + (y - b)² = R²

Если центр в начале координат (0, 0), то:
x² + y² = R²

⸻

Дробно-рациональные неравенства. Метод интервалов.
Это неравенства, где переменная находится в знаменателе.

Пример: (2x - 3)/(x + 1) > 0

Метод интервалов:
 1. Найти ОДЗ (исключить значения, при которых знаменатель равен 0).
 2. Привести к виду f(x)/g(x) > 0 или < 0.
 3. Найти нули числителя и знаменателя.
 4. Разбить числовую прямую на интервалы.
 5. Определить знак выражения на каждом интервале.
 6. Выбрать подходящие интервалы.

Пример:
(x - 2)/(x + 3) ≥ 0
ОДЗ: x ≠ -3
Нули: x = 2
Точки: -3 и 2
Интервалы: (-∞, -3), (-3, 2), (2, +∞)
 • при x < -3 → дробь > 0
 • при -3 < x < 2 → дробь < 0
 • при x > 2 → дробь > 0

Решение: (-∞, -3) ∪ [2, +∞)

1. Дробно-рациональные уравнения

Пример 1.
(3x + 2)/(x - 1) = 4

Решение:
ОДЗ: x ≠ 1
Умножим обе части на (x - 1):
3x + 2 = 4(x - 1)
3x + 2 = 4x - 4
 • x = -4 - 2 = -6
x = 6

Проверка: x = 6 ∈ ОДЗ → подходит.

Ответ: x = 6

⸻

2. Системы нелинейных уравнений

Пример 2.
{
x² + y² = 25
y = x + 1
}

Решение:
Подставим y = x + 1 в первое уравнение:
x² + (x + 1)² = 25
x² + x² + 2x + 1 = 25
2x² + 2x - 24 = 0
x² + x - 12 = 0
(x + 4)(x - 3) = 0

x₁ = -4, x₂ = 3

Для x₁ = -4 → y = -3
Для x₂ = 3 → y = 4

Ответ: (-4, -3), (3, 4)

⸻

3. Длина отрезка и уравнение окружности

Пример 3.
Найти длину отрезка AB, если A(1, 2), B(5, 5).

Решение:
AB = √((5 - 1)² + (5 - 2)²)
AB = √(4² + 3²) = √(16 + 9) = √25 = 5

Ответ: AB = 5

⸻

Пример 4.
Найти уравнение окружности с центром в O(2, -1) и радиусом R = 3.

Решение:
(x - 2)² + (y + 1)² = 9

Ответ: (x - 2)² + (y + 1)² = 9

⸻

4. Дробно-рациональные неравенства (метод интервалов)

Пример 5.
(x - 1)/(x + 2) > 0

Решение:
ОДЗ: x ≠ -2
Нули числителя: x = 1
Точки: -2 и 1

Интервалы: (-∞, -2), (-2, 1), (1, +∞)

Проверим знаки:
 • при x < -2 → числитель < 0, знаменатель < 0 → дробь > 0
 • при -2 < x < 1 → числитель < 0, знаменатель > 0 → дробь < 0
 • при x > 1 → числитель > 0, знаменатель > 0 → дробь > 0

Знаки «> 0» выполняются на: (-∞, -2) ∪ (1, +∞)

Ответ: x ∈ (-∞, -2) ∪ (1, +∞)
