Арифметическая и геометрическая прогрессия

⸻

Арифметическая прогрессия (АП)

Определение:
Арифметическая прогрессия — это последовательность чисел, в которой каждое следующее число получается прибавлением одного и того же числа (разности d) к предыдущему.

Формула n-го члена:
aₙ = a₁ + (n - 1) * d

Формула суммы первых n членов:
Sₙ = (n / 2) * (a₁ + aₙ)
или
Sₙ = (n / 2) * (2a₁ + (n - 1) * d)

Пример:
Найти сумму первых 6 членов прогрессии: 2, 5, 8, 11, …
 • a₁ = 2, d = 3
 • a₆ = 2 + (6 - 1) * 3 = 17
 • S₆ = (6 / 2) * (2 + 17) = 3 * 19 = 57

Ответ: S₆ = 57

⸻

Геометрическая прогрессия (ГП)

Определение:
Геометрическая прогрессия — это последовательность чисел, в которой каждое следующее число получается умножением предыдущего на одно и то же число (знаменатель прогрессии q).

Формула n-го члена:
bₙ = b₁ * q^(n - 1)

Формула суммы первых n членов (если q ≠ 1):
Sₙ = b₁ * (1 - qⁿ) / (1 - q)
или
Sₙ = b₁ * (qⁿ - 1) / (q - 1)

Пример:
Найти сумму первых 4 членов прогрессии: 3, 6, 12, 24, …
 • b₁ = 3, q = 2
 • S₄ = 3 * (2⁴ - 1) / (2 - 1) = 3 * 15 = 45

Ответ: S₄ = 45

⸻

Бесконечно убывающая геометрическая прогрессия

Определение:
Если знаменатель геометрической прогрессии удовлетворяет условию |q| < 1, то прогрессия называется бесконечно убывающей.

В этом случае сумма всех её членов конечна и вычисляется по формуле:

S = b₁ / (1 - q)

где b₁ — первый член прогрессии, q — знаменатель.

Пример:
Найти сумму бесконечно убывающей прогрессии: 8, 4, 2, 1, …
 • b₁ = 8, q = 1/2
 • S = 8 / (1 - 1/2) = 8 / (1/2) = 16

Ответ: S = 16

⸻

Где применяются прогрессии
 • В финансах: проценты, кредиты, вклады.
 • В физике: движение с постоянным ускорением (АП), радиоактивный распад (ГП).
 • В строительстве: равномерное размещение ступенек, балок, колонн.
 • В информатике: вычислительная сложность алгоритмов (например, бинарный поиск — ГП).
 • В реальной жизни: задачи на рост или уменьшение величин (например, каждый день прибавляют по 5 минут — АП; каждый месяц уменьшается значение вдвое —ГП).

Арифметическая прогрессия (АП)

Формула n-го члена:
aₙ = a₁ + (n-1)*d

Сумма n членов:
Sₙ = n/2 * (2*a₁ + (n-1)*d)

⸻

Пример 1. Найти n-й член АП

Дано:
a₁ = 3, d = 5. Найти 10-й член a₁₀.

Решение:
a₁₀ = 3 + (10-1)5 = 3 + 95 = 3 + 45 = 48

Ответ: a₁₀ = 48

⸻

Пример 2. Найти сумму n членов АП

Дано:
a₁ = 2, d = 3, n = 20

Решение:
S₂₀ = 20/2 * (2*2 + (20-1)*3) = 10 * (4 + 57) = 10 * 61 = 610

Ответ: S₂₀ = 610

⸻

Геометрическая прогрессия (ГП)

Формула n-го члена:
bₙ = b₁ * q^(n-1)

Сумма n членов (q ≠ 1):
Sₙ = b₁ * (q^n - 1)/(q - 1)

⸻

Пример 3. Найти n-й член ГП

Дано:
b₁ = 2, q = 3. Найти b₆.

Решение:
b₆ = 2 * 3^(6-1) = 2 * 3^5 = 2 * 243 = 486

Ответ: b₆ = 486

⸻

Пример 4. Найти сумму n членов ГП

Дано:
b₁ = 1, q = 2, n = 8

Решение:
S₈ = 1 * (2^8 - 1)/(2 - 1) = (256 - 1)/1 = 255

Ответ: S₈ = 255

⸻

Бесконечно убывающая геометрическая прогрессия

Формула суммы бесконечной прогрессии (при |q| < 1):
S = b₁ / (1 - q)

⸻

Пример 5. Найти сумму бесконечной ГП

Дано:
b₁ = 8, q = 1/2

Решение:
S = 8 / (1 - 1/2) = 8 / (1/2) = 16

Ответ: S = 16
