Теорема синусов, теорема косинусов, формула Герона

⸻

• Теорема синусов

В любом треугольнике ABC:

a / sin A = b / sin B = c / sin C = 2R,

где:
 • a, b, c — стороны треугольника,
 • A, B, C — углы, противоположные этим сторонам,
 • R — радиус описанной окружности треугольника.

Применение:
 • Решение треугольников, если известны:
 • две стороны и угол, не заключённый между ними (SAS),
 • две угла и одна сторона (AAS или ASA).

Пример:
Дано: A = 30°, B = 45°, a = 10.
Найти сторону b:
b / sin 45° = 10 / sin 30° → b = 10 * sin 45° / sin 30° ≈ 14,14

⸻

• Теорема косинусов

В любом треугольнике ABC:

a² = b² + c² - 2bc * cos A
b² = a² + c² - 2ac * cos B
c² = a² + b² - 2ab * cos C

Применение:
 • Решение треугольников, если известны:
 • две стороны и угол между ними (SAS),
 • все три стороны (SSS) для нахождения углов.

Пример:
Дано: b = 5, c = 7, A = 60°.
Найти сторону a:
a² = 5² + 7² - 257cos 60° = 25 + 49 - 700,5 = 74 - 35 = 39
a ≈ 6,24

• Формула Герона

Если известны все три стороны треугольника a, b, c, площадь S можно найти по формуле:

S = √(p(p-a)(p-b)(p-c)),

где p = (a + b + c)/2 — полупериметр.

Пример:
Дано: a = 5, b = 6, c = 7
p = (5 + 6 + 7)/2 = 9
S = √(9*(9-5)(9-6)(9-7)) = √(943*2) = √216 ≈ 14,7

Применение:
 • Вычисление площади треугольника по трём сторонам,
 • Используется при решении треугольников, когда углы неизвестны.

• Пример 1. Теорема синусов
Дано: треугольник ABC, A = 30°, B = 45°, a = 10.
Найти сторону b.

Решение:
 1. Применяем теорему синусов: b / sin B = a / sin A
 2. Подставляем значения: b / sin 45° = 10 / sin 30°
 3. b = 10 * sin 45° / sin 30° ≈ 10 * 0,7071 / 0,5 ≈ 14,14

Ответ: b ≈ 14,14

⸻

• Пример 2. Теорема синусов
Дано: треугольник ABC, a = 8, b = 6, угол A = 60°.
Найти угол B.

Решение:
 1. a / sin A = b / sin B → sin B = b * sin A / a
 2. sin B = 6 * sin 60° / 8 = 6 * 0,8660 / 8 ≈ 0,6495
 3. B ≈ arcsin(0,6495) ≈ 40,5°

Ответ: B ≈ 40,5°

⸻

• Пример 3. Теорема косинусов
Дано: треугольник ABC, b = 5, c = 7, угол A = 60°.
Найти сторону a.

Решение:
 1. Применяем теорему косинусов: a² = b² + c² - 2bc*cos A
 2. a² = 5² + 7² - 257cos 60° = 25 + 49 - 700,5 = 39
 3. a ≈ √39 ≈ 6,24

Ответ: a ≈ 6,24

⸻

• Пример 4. Теорема косинусов (нахождение угла)
Дано: треугольник с сторонами a = 8, b = 6, c = 5.
Найти угол A.

Решение:
 1. cos A = (b² + c² - a²) / (2bc)
 2. cos A = (6² + 5² - 8²) / (265) = (36 + 25 - 64) / 60 = (-3)/60 = -0,05
 3. A ≈ arccos(-0,05) ≈ 93°

Ответ: A ≈ 93°

⸻

• Пример 5. Формула Герона
Дано: треугольник с сторонами a = 5, b = 6, c = 7.
Найти площадь.

Решение:
 1. Полупериметр: p = (a + b + c)/2 = (5 + 6 + 7)/2 = 9
 2. S = √(p*(p-a)(p-b)(p-c)) = √(9*(9-5)(9-6)(9-7)) = √(943*2) = √216 ≈ 14,7

Ответ: S ≈ 14,7

⸻

• Пример 6. Формула Герона (треугольник с дробными сторонами)
Дано: a = 4, b = 6, c = 5.5
Найти площадь.

Решение:
 1. p = (4 + 6 + 5.5)/2 = 15.5/2 = 7.75
 2. S = √(7.75*(7.75-4)(7.75-6)(7.75-5.5)) = √(7.753.751.75*2.25)
 3. S ≈ √114.04 ≈ 10.68

Ответ: S ≈ 10.68
