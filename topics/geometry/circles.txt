Вписанные и описанные окружности

⸻

• Описанная окружность треугольника

Определение:
Описанная окружность — это окружность, которая проходит через все вершины треугольника.

Свойства:
 • Центр описанной окружности — точка пересечения серединных перпендикуляров к сторонам треугольника.
 • Радиус описанной окружности R можно найти по формуле:
R = (abc) / (4S),
где a, b, c — стороны треугольника, S — площадь треугольника.

⸻

• Вписанная окружность треугольника

Определение:
Вписанная окружность — это окружность, которая касается всех сторон треугольника.

Свойства:
 • Центр вписанной окружности — точка пересечения биссектрис углов треугольника.
 • Радиус вписанной окружности r можно найти по формуле:
r = S / p,
где S — площадь треугольника, p = (a + b + c)/2 — полупериметр.

⸻

• Прямоугольный треугольник и его окружности
 • Вписанная окружность: радиус r = (a + b - c)/2,
где a и b — катеты, c — гипотенуза.
 • Описанная окружность: радиус R = c/2,
где c — гипотенуза.

Пример:
Прямоугольный треугольник с катетами 3 и 4, гипотенуза 5:
 • r = (3 + 4 - 5)/2 = 1
 • R = 5/2 = 2,5

⸻

• Вписанные и описанные четырёхугольники

Вписанный четырёхугольник:
 • Все вершины лежат на одной окружности.
 • Углы, лежащие напротив друг друга, в сумме равны 180°.

Описанный четырёхугольник:
 • Все стороны касаются одной окружности.
 • Сумма противоположных сторон равна: a + c = b + d

Применение:
 • Используется для вычисления площадей и радиусов окружностей,
 • Для доказательства свойств четырёхугольников.

• Пример 1. Прямоугольный треугольник и его окружности
Дано: прямоугольный треугольник с катетами a = 6 и b = 8, гипотенуза c = 10.
Найти радиусы вписанной и описанной окружностей.

Решение:
 1. Вписанная окружность: r = (a + b - c)/2 = (6 + 8 - 10)/2 = 4/2 = 2
 2. Описанная окружность: R = c/2 = 10/2 = 5

Ответ: r = 2, R = 5

⸻

• Пример 2. Вписанный четырёхугольник
Дано: четырёхугольник ABCD вписанный в окружность. Углы A и C равны 70° и 110° соответственно. Найти сумму углов B и D.

Решение:
 • В вписанном четырёхугольнике сумма противоположных углов равна 180°:
A + C = 180° → 70° + 110° = 180°
 • Следовательно, B + D = 180°

Ответ: B + D = 180°

⸻

• Пример 3. Описанный четырёхугольник
Дано: четырёхугольник ABCD, описанный около окружности, со сторонами a = 6, b = 4, c = 8. Найти сторону d.

Решение:
 • Для описанного четырёхугольника: a + c = b + d → 6 + 8 = 4 + d → 14 = 4 + d → d = 10

Ответ: d = 10
